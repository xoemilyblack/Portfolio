My Projects

Arduino Controlled Mirror

This was a project I made in my first year of college for an innovation showcase competition. It is a mirror controlled by an arduino computer that connects to an LED light strip inside a special frame. This frame has a real mirror in the back and a one way mirror on the front. When lit up it gives the illusion as if it goes on forever. It won second place !

Regression Testing Tool

During my first co-op I worked at a company that developed software for travel agencies called Global Matrix. My major project was to make a java program that would take any schemas of xml file (from different travel agencies) and process all the elements and sort the output by which travel agency it initially came from. This reduced the time it took to complete this task that was once manual.

API Website Design

Also at my co-op I recieved a project to re-design their API website. It was a single page that would go to the appropriate section when a link in the navigation was clicked. All paths to images were relative so it could be opened anywhere. I designed a new logo in the header and changed the whole colour schemas.
